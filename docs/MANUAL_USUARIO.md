## Introducción

MarketWorld es un sistema de gestión empresarial que permite administrar las operaciones de un negocio desde una interfaz web intuitiva.

### Requisitos

- Navegador web actualizado (Chrome, Firefox, Edge o Safari)
- Conexión a internet para cargar recursos externos
- Resolución de pantalla mínima recomendada: 1024x768

---

## Inicio de Sesión

### Acceder al sistema

1. Abre el navegador web
2. Ingresa a la página de inicio de sesión
3. Escribe tu correo electrónico
4. Escribe tu contraseña
5. Haz clic en **Iniciar Sesión**

### Opciones adicionales

- **Recordarme**: Mantiene tu sesión activa
- **Olvidé mi contraseña**: Recupera el acceso a tu cuenta

### Solución de problemas

| Problema | Solución |
|----------|----------|
| Credenciales incorrectas | Verifica que el correo y contraseña sean correctos |
| Cuenta bloqueada | Contacta al administrador del sistema |
| Página no carga | Verifica tu conexión a internet |

## Panel Principal

Después de iniciar sesión, verás la página de inicio con:

- **Menú lateral**: Acceso a todos los módulos
- **Barra superior**: Búsqueda, notificaciones y perfil
- **Área de contenido**: Información del módulo activo

### Navegación

Usa el menú lateral para acceder a:

- Inicio
- Dashboard
- Inventario
- Facturación
- Contabilidad
- Compras
- CRM
- Reportes
- Configuración

## Dashboard

El panel de control muestra los indicadores clave del negocio.

### Elementos del Dashboard

1. **Tarjetas de resumen**
   - Ventas del período
   - Facturas pendientes
   - Productos en stock bajo
   - Clientes nuevos

2. **Gráficos**
   - Ventas por período
   - Productos más vendidos
   - Estado de facturas

3. **Calendario**
   - Eventos y recordatorios
   - Vencimientos de facturas

### Filtros de fecha

Selecciona el período a analizar usando los campos de fecha en la parte superior.

## Inventario

Gestión completa de productos y stock.

### Ver productos

1. Accede a **Inventario** desde el menú
2. Visualiza la lista de productos
3. Usa los filtros para buscar productos específicos

### Agregar producto

1. Haz clic en **Nuevo Producto**
2. Completa la información:
   - SKU (código único)
   - Nombre
   - Descripción
   - Categoría
   - Precio de compra
   - Precio de venta
   - Stock inicial
   - Stock mínimo
3. Haz clic en **Guardar**

### Editar producto

1. Busca el producto en la lista
2. Haz clic en el botón de editar
3. Modifica la información
4. Haz clic en **Guardar cambios**

### Movimientos de inventario

Registra entradas, salidas y ajustes de stock:

1. Selecciona el producto
2. Indica el tipo de movimiento
3. Ingresa la cantidad
4. Agrega una descripción
5. Confirma el movimiento

## Facturación

Sistema de punto de venta y emisión de facturas.

### Crear factura

1. Accede a **Facturación**
2. Selecciona o busca un cliente
3. Agrega productos al carrito
4. Verifica los totales
5. Selecciona el método de pago
6. Haz clic en **Generar Factura**

### Buscar facturas

1. Usa la barra de búsqueda
2. Filtra por estado, fecha o cliente
3. Haz clic en una factura para ver detalles

### Estados de factura

| Estado | Descripción |
|--------|-------------|
| Pendiente | Factura emitida, pago no recibido |
| Pagada | Pago recibido completo |
| Vencida | Pasó la fecha de vencimiento |
| Anulada | Factura cancelada |

## Contabilidad

Registro y seguimiento de transacciones financieras.

### Funciones disponibles

- Ver resumen de ingresos y gastos
- Registrar transacciones
- Consultar estados financieros
- Generar balances

### Registrar transacción

1. Haz clic en **Nueva Transacción**
2. Selecciona el tipo (ingreso o gasto)
3. Ingresa el monto
4. Selecciona la categoría
5. Agrega descripción
6. Guarda la transacción

---

## Compras

Gestión de proveedores y órdenes de compra.

### Gestionar proveedores

1. Ve a la sección de proveedores
2. Agrega, edita o desactiva proveedores
3. Consulta el historial de compras

### Crear orden de compra

1. Selecciona un proveedor
2. Agrega productos a la orden
3. Ingresa cantidades y precios
4. Revisa el total
5. Genera la orden

### Estados de orden

| Estado | Descripción |
|--------|-------------|
| Pendiente | Orden enviada, esperando productos |
| Recibido | Productos recibidos en inventario |
| Pagado | Orden pagada al proveedor |
| Cancelado | Orden anulada |

---

## CRM

Administración de relaciones con clientes.

### Gestionar clientes

1. Accede a **CRM**
2. Visualiza la lista de clientes
3. Usa filtros por segmento o estado

### Agregar cliente

1. Haz clic en **Nuevo Cliente**
2. Completa los datos:
   - Nombre
   - Documento
   - Tipo de documento
   - Correo electrónico
   - Teléfono
   - Dirección
3. Guarda el cliente

### Segmentos de clientes

| Segmento | Descripción |
|----------|-------------|
| Nuevo | Cliente recién registrado |
| Frecuente | Cliente con compras regulares |
| Premium | Cliente de alto valor |
| Corporativo | Clientes empresariales |

## Reportes

Generación de informes y análisis.

### Tipos de reportes

- **Ventas**: Resumen de ventas por período
- **Inventario**: Estado del stock
- **Financiero**: Balance de ingresos y gastos
- **Clientes**: Análisis de segmentación

### Generar reporte

1. Accede a **Reportes**
2. Selecciona el tipo de reporte
3. Configura el período
4. Haz clic en **Generar**
5. Visualiza los gráficos y tablas

---

## Configuración

Administración del sistema y usuarios.

### Gestionar usuarios

1. Ve a **Configuración**
2. Accede a la sección de usuarios
3. Agrega, edita o desactiva usuarios

### Roles disponibles

| Rol | Permisos |
|-----|----------|
| Administrador | Acceso completo al sistema |
| Gerente | Acceso a reportes y configuración |
| Vendedor | Acceso a ventas e inventario |
| Soporte | Acceso de solo lectura |

### Cambiar contraseña

1. Ve a tu perfil
2. Selecciona **Cambiar contraseña**
3. Ingresa la contraseña actual
4. Ingresa la nueva contraseña
5. Confirma la nueva contraseña
6. Guarda los cambios

## Preguntas Frecuentes

### ¿Cómo recupero mi contraseña?

Haz clic en "¿Olvidaste tu contraseña?" en la página de inicio de sesión y sigue las instrucciones.

### ¿Puedo usar el sistema en dispositivos móviles?

Sí, el sistema tiene diseño responsivo que se adapta a diferentes tamaños de pantalla.

### ¿Cómo exporto un reporte?

En la sección de reportes, genera el informe y usa el botón de exportar para descargarlo.

### ¿Quién puede crear usuarios nuevos?

Solo los usuarios con rol de Administrador pueden crear y gestionar otros usuarios.

### ¿Dónde veo el historial de cambios?

Cada módulo tiene un registro de actividad que muestra las modificaciones realizadas.
