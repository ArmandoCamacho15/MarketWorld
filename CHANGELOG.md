# Registro de Cambios

Todos los cambios importantes de este proyecto se documentan en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/).

---

## [1.0.0] - 2025-12-01

### Agregado

- Sistema de autenticación con inicio de sesión
- Registro de nuevos usuarios
- Panel de control (Dashboard) con indicadores
- Módulo de gestión de inventario
- Sistema de facturación y punto de venta
- Módulo de contabilidad
- Gestión de compras y proveedores
- CRM para administración de clientes
- Generación de reportes con gráficos
- Configuración de usuarios y permisos
- Menú lateral responsivo
- Esquema de base de datos MySQL
- Documentación del proyecto

### Características técnicas

- Diseño responsivo con Bootstrap 5.3
- Gráficos interactivos con Chart.js
- Calendario con FullCalendar
- Validación de formularios en JavaScript
- Almacenamiento de sesión en localStorage
